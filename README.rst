Easy tool for using Ethereum Smart Contracts
###############################################################
  
Installation
====================

.. code-block:: bash

    $ git clone https://github.com/simon-liu/easy-contract
    $ cd easy-contract && python setup.py install

Usage
====================

.. code-block:: bash

    $ easy_contract /path/to/build/contracts/DemoToken.json
